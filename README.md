Basic Calculator in Python : 
This is a simple console-based calculator program written in Python. It allows users to perform basic mathematical operations on two numbers.

Features : 
1. Addition: Adds two numbers.
2. Subtraction: Subtracts the second number from the first.
3. Multiplication: Multiplies two numbers.
4. Division: Divides the first number by the second (includes safety checks to avoid division by zero).

Error Handling : Warns the user if they enter an option outside the 1–4 range or attempt to divide by zero.

How It Works :
1. The program displays a numbered menu of options (1 to 4).
2. The user inputs their chosen menu number.
3. The user enters two numbers (supports decimal values).
4. The program checks the selected option using if-elif-else statements and prints the result.
